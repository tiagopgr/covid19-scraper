# covid19-scraper :gear:

Scraper de dados do covid-19 do site https://covid.saude.gov.br/ com informações sobre os dados do **Brasil** e **Ceará**.

Tecnologias usadas:

-   NodeJS
-   MongoDB

### Rodando a aplicação

Criar o arquivo .env usando como base o arquivo .env.example

```shell
$ cp .env.example .env
```

```shell
$ npm install
$ yarn run exec
# ou
$ npm run exec
```
